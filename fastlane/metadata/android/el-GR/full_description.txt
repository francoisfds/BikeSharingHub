<p>
Το BikeSharingHub εμφανίζει τη διαθεσιμότητα κοινόχρηστων ποδηλάτων μεταξύ εκατοντάδων πόλεων στην ίδια εφαρμογή.
</p>
<p>
Τα δεδομένα αντλούνται από το API του CityBik.es, το οποίο συγκεντρώνει ανοικτά δεδομένα που παρέχονται από τους φορείς εκμετάλλευσης διαθέσιμων κοινόχρηστων ποδηλάτων. Αυτό το API είναι ανοικτού κώδικα και η εφαρμογή επιτρέπει τον ορισμό άλλης διεύθυνσης API από την προεπιλεγμένη.
</p>
<p>
Βασίζεται στην εφαρμογή OpenBikeSharing (repo «F-Droid Archive»).
</p>
